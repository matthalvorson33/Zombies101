The system shall display the user’s high score in the top left corner of the screen while the user is playing.

The system shall display a title screen with clickable ‘start’ and ‘instructions’ buttons when the game is launched.

The system shall save the user’s game data every time the user crosses a checkpoint flag while playing.

The system shall pause the current game and display a menu that contains one clickable ‘exit’ button when the user presses the ‘escape’ key.

The system shall display an instructions page* if the user clicks on the ‘instructions’ button from the title page.

*requirements for the instructions page would be explained in a different FR
